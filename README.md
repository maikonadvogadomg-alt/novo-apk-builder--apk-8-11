# APK Builder

## Descrição
O APK Builder é uma ferramenta que converte PWAs (Progressive Web Apps) em APKs (Android Package) nativos. Ele permite importar projetos de arquivos ZIP ou diretamente do GitHub e gera o projeto Android necessário.

## Tecnologias
- **Front-end**: React, Vite
- **Back-end**: Node.js, Express
- **Banco de Dados**: PostgreSQL (Neon DB)
- **Autenticação**: JWT

## Instalação

### Pré-requisitos
- Node.js (v16 ou superior)
- NPM ou Yarn
- Uma conta no Neon DB (se utilizar o banco de dados)
